# Cron

A way to schedule a task to run automatically at fixed times — every 5 minutes, every hour, every day at 09:00 Malta time, etc. The platform running the code (e.g. [[Railway]] or GitHub Actions) takes care of starting the task on time. The audit lists each TRC cron job in [[Master Audit]] § 4.
